# 42 Program Monitoring System – Quick Overview

This repo includes resources related to the **42 Next** pedagogical monitoring system, which replaces the old Paces, deadlines ect ...

## How it works

The Common Core is divided into **8 Milestones (M1–M8)**, each with a reference duration based on studying **40 hours/week**. The full program has a reference duration of **1 year** and a maximum of **2 years**.

Your **Completion** is calculated from:
- The milestones you've validated
- Your weekly "effort" (attendance + peer reviews)

Each week, the system compares your completion to the **Reference Completion** and gives you one of 5 statuses:
- Outstanding Pace
- Great Pace
- You're On Track
- You can do  better than that
- Significant Lag

## Watch out: Significant Lag

If your completion falls to **less than half** the reference completion, a **28-day Grace Period** starts. If you don't recover your status within those 28 days, you'll be dropped from the program.

## Days Off

You earn **2 days off per month** (10 days/year total). Using them pauses the program clock and can even pause a Grace Period.
